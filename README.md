# 🏋️ EliteAthlete — Sports Training & Performance Platform

> A fully responsive, multi-page web application dedicated to athletic training, nutrition, and peak performance.

---

## 📌 Project Overview

**EliteAthlete** is a professional web platform built for athletes, fitness enthusiasts, and coaches across the globe. It provides structured training programs, sport-specific nutrition advice, expert coach profiles, an interactive gallery, and the latest fitness news — all in one clean, modern interface.

## 📄 Pages

Page Description 
`index.html`Home — hero banner, featured athletes, quick highlights  `training.html` Training Programs — strength, cardio, sport-specific plans 
`nutrition.html` Nutrition — meal plans, macros guide, hydration tips 
`coaches.html`  Coaches — expert profiles, bios, and specializations 
`blog.html`  Blog/News — training tips, athlete stories, articles `gallery.html`  Gallery — filterable photo gallery by sport category  `faq.html` FAQ — common training and nutrition questions answered 
`contact.html` Contact — booking form with validation, map, social links

---

## ⚙️ Technologies Used

- **HTML5** — Semantic, accessible markup
- **CSS3** — Custom styles, Flexbox, media queries
- **JavaScript** — Interactivity and dynamic features
- **Bootstrap 5** — Responsive grid, navbar, modals, components


## ⚡ Interactive Features

1. **Fitness Calculator** — BMI and daily calorie needs calculator
2. **Filterable Gallery** — Filter training photos by sport type
3. **Dark / Light Mode Toggle** — Site-wide theme switcher
